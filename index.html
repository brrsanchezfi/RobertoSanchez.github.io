¡Hola! Soy Roberto Sánchez, ingeniero electricista apasionado por la programación, la ciencia de datos y el desarrollo de software. Con una sólida formación técnica, combino mi conocimiento en ingeniería con mi amor por el software para crear soluciones innovadoras. Java

Enfoque principal en programación orientada a objetos (POO) y bases de datos.

Matlab

Bases de datos

Experiencia sólida en el manejo de MySQL y SQLite, con un dominio básico en la gestión de DDL, DML y DCL.

POO

Analisis de datos
